# Setup

## From scratch

```bash
  git clone https://github.com/N0ku/Idle-Game.git
```

## From project directory

In 2 different terminals

```bash
  cd server
  npm install
  npm run dev
```

```bash
  cd client
  npm install
  npm run dev
```

#Lauch Docker
